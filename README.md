# 侠客服女少东家 Codex 桌宠

一个非官方同人 Codex/Petdex 桌宠，灵感来自 *Where Winds Meet* / 《燕云十六声》中黑白初始侠客服风格的女少东家形象。

本仓库包含一个 Codex 兼容的动态桌宠包：

- `pet.json`
- `spritesheet.webp`

预览：

![Contact sheet](docs/contact-sheet.png)

## 使用方式

可以把这个文件夹复制到 Codex 的 pets 目录中使用；如果想分享到 Petdex，也可以上传这个文件夹或 ZIP 包。

桌宠包的基础结构如下：

```text
pet.json
spritesheet.webp
```
